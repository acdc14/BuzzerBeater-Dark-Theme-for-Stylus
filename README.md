# BuzzerBeater Dark Theme for Stylus

Dark mode theme pack for the website **BuzzerBeater** using the Stylus browser extension.

This project modernizes the visual appearance of the site while keeping compatibility with the original legacy structure.

---

# Features

- Dark mode support
- Multiple themes
- CSS variables based system
- Progressive legacy-safe overrides
- Easy customization
- Compatible with Stylus

---

# Requirements

Install the Stylus browser extension:

https://github.com/openstyles/stylus

---

# Installation

1. Install Stylus
2. Create a new style
3. Copy the CSS from:

```text
styles.css
```

4. Save
5. Reload BuzzerBeater

---

# Included Themes

- Blue Theme
- Grey Theme

Themes are detected automatically based on the original BuzzerBeater styles.

---

# Customization

The project uses CSS variables, making it easy to create or modify themes.

Example:

```css
--bg
--surface-1
--surface-2
--primary
--text
--accent
```

You can also customize semantic colors:

```css
--score-win
--match-friendly
```

---

# Notes

Because BuzzerBeater uses a legacy structure:

- Some pages may still require manual fixes
- Some inline styles may override the theme
- Changes are applied progressively to minimize visual issues
